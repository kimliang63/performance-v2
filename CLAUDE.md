# ghr-performance

## 是什么
GHR 绩效管理 + 流程引擎平台的纯 HTML 原型项目

## 技术栈
纯 HTML 原型（HTML + CSS + JS，无框架）
- 不使用 React / Vue / Ant Design 等框架
- Mock 数据内置在 HTML 中
- 每个版本功能闭环，可独立演示

## 原型开发规则
- 一套代码越滚越大，不按版本分目录
- 版本确认后快照到 `confirmed/vN/`，后续不改
- 改造已有功能必须在 CHANGELOG.md 记录
- Git 不是必须的，目录结构即版本控制
- **废弃所有旧 React 原型**，全部重新输出纯 HTML 原型

## 边界
- 流程引擎平台：流程设计器、流程管理、表单管理、消息中心
- 绩效管理：基础配置、绩效方案、绩效活动、绩效流程
