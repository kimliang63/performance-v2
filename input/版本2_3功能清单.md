### 3.5 流程模型配置清单

#### 3.5.1 目标制定流程

```
流程节点：
├── 发起人节点（被考核人）
│   ├── 表单：PBC目标制定表单
│   └── 操作：填写目标 → 提交
├── 审批人节点（考核人）
│   ├── 表单：PBC目标制定表单（只读）
│   ├── 操作：同意/不同意/驳回到发起人
│   ├── 审批人设置：按汇报关系（直接上级）
│   └── 多人审批：否
└── 条件分支
    ├── 条件1：审批通过 → 结束
    └── 条件2：驳回 → 返回发起人重新填写

抄送：考核人审批通过后抄送HRBP
```

#### 3.5.2 绩效考核流程

```
流程节点：
├── 办理人节点（考核人）
│   ├── 表单：绩效考核表单
│   └── 操作：评分/填写评语 → 提交
└── 结束

说明：考核人逐个处理各被考核人的绩效评分，无分支
```

#### 3.5.3 结果审定流程

```
流程节点：
├── 审批人节点（审定人）
│   ├── 表单：结果审定表单
│   ├── 操作：确认/调整绩效结果
│   ├── 审批人设置：按矩阵角色（部门负责人）
│   ├── 多人审批：会签
│   └── 调整后结果：需填写调整原因
└── 结束

说明：多个审定人会签，全部同意后结束
```

### 3.6 流程节点类型对照表

| 绩效场景节点 | 对应流程引擎节点 | 多人审批支持 | 审批操作 |
|-------------|----------------|------------|---------|
| 目标填写 | 办理人节点 | 否 | 提交 |
| 目标审批 | 审批人节点 | 否 | 同意/不同意/驳回 |
| 绩效考核评分 | 办理人节点 | 否 | 评分 |
| 结果审定 | 审批人节点 | 是（会签） | 确认/调整 |
| 员工确认 | 办理人节点 | 否 | 确认/驳回 |
| 抄送节点 | 抄送人节点 | — | 仅抄送 |
