核心数据库表

表名 说明 层级 关键字段
perf_scheme 绩效方案 V2 方案 version, status, grade_rule_id
perf_assessment_group 考核组 V2 方案 scheme_id, rule_type, rule_config
perf_stage_config 环节配置 V2 方案 stage_type, flow_template_id, form_template_id
perf_activity 考核活动 V4 活动 scheme_version, scheme_snapshot (JSON)
perf_activity_participant 活动参与人 V4 活动 stage_status (JSON), final_grade
perf_scheme_version 方案版本历史 版本管理 scheme_id, version, snapshot (JSON)
perf_activity_config_override 活动配置覆盖 配置覆盖 config_type, config_key, config_value (JSON)
goal_detail 目标明细表 环节数据 status (active/voided), modules (JSON)
eval_detail 考核明细表 环节数据 score_data (JSON), total_score
ratification_detail 审定明细表 环节数据 adjusted_grade, adjust_reason
final_result 绩效结果表 结果汇总 final_grade, final_score, locked
