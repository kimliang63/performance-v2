层级1：单个指标得分
每个指标用自己的衡量标准 grades，公式引用该指标的参考值
层级2：总分汇总
多个指标得分 + 权重 → 加权求和 / 自定义公式

选择指标 → 该指标的衡量标准 grades 注入为公式变量

📦 可用字段 · 指标A
💡 点击左侧字段插入 · 公式变量来自当前指标的衡量标准 grades ✓ 公式校验通过

总分 = Σ(各指标得分 × 权重) + 加分 - 减分

总分公式引用的字段

  • indicator_score_A — 指标A的得分（层级1算出）
  • indicator_score_B — 指标B的得分
  • indicator_score_C — 指标C的得分
  • weight_A / weight_B / weight_C — 各指标权重
  • bonus — 加分
  • penalty — 减分

关键区别

  • 层级1：引用的是衡量标准 grades（挑战值/目标值/保底值）
  • 层级2：引用的是各指标的得分结果（已算出的分数）
  • 层级2 不直接引用 grades，而是引用层级1的计算结果
  • 两个层级的公式独立配置，层级2的输入 = 层级1的输出
