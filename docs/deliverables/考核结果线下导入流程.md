# 流程：考核结果线下导入流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 考核结果线下导入流程 |
| 流程编码 | flow_performance_review_import |
| 流程说明 | HRBP批量导入员工绩效考核结果，经双线上级→直接上级审批确认 |
| 适用场景 | 线下考核结果批量导入（Excel导入），不走员工自评节点 |

### 流程节点

```
开始 → HRBP导入 → 双线上级评分（可跳过）→ 直接上级评分 → 结束
```

## 五、与线上考核流程的差异

| 差异项 | 线上考核流程（Flow 03） | 考核结果线下导入（本流程） |
|--------|----------------------|------------------------|
| 起始节点 | 被考核人自评 | HRBP导入Excel |
| 员工参与 | 员工填写完成值+自评 | 不走员工自评节点 |
| 导入方式 | 无批量导入 | Excel批量导入 |
| 审批链 | 间接上级→直接上级 | 双线上级→直接上级 |
| 适用场景 | 线上逐人考核 | 线下已有考核结果，批量导入 |
