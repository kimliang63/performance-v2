# 流程3：线上考核流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 线上考核流程 |
| 流程编码 | flow_performance_review_online |
| 流程说明 | 被考核人自评→间接上级评分→直接上级评分 |
| 适用场景 | 线上考核（员工自评+上级评分） |

### 流程节点

```
开始 → 被考核人自评分 → 间接上级评分（可跳过）→ 直接上级评分 → 结束
```
