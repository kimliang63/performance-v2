## 使用说明

1. **复制本模板**，将 `{占位符}` 替换为实际内容
2. **功能模块可重复**：每个功能模块按「模块元信息 → 权限表 → 功能说明表」三件套组织
3. **功能说明表**的「操作/处理逻辑」列建议用有序列表，层级清晰
4. **截图/原型**：在「用户界面」列嵌入对应原型截图或链接
5. **版本管理**：多版本文档中，用二级标题 `### V{N}：{版本名}` 分组功能模块
