# 强制分布规则 — 字段清单与交互规则

> 基于 `perf-forced-distribution.html` 原型提取，版本 v3 快照

---

## 1. 页面布局

```
┌─────────────────────────────────────────────────────────┐
│ Header: Logo + 标题"强制分布规则" + 返回导航 + 新建规则 │
├─────────────────────────────────────────────────────────┤
│ Section 1: 等级定义                                     │
│   └─ 表格: 等级名称/代码/分数区间/颜色/排序/操作        │
├─────────────────────────────────────────────────────────┤
│ Section 2: 分布比例配置                                 │
│   ├─ 表格: 等级/要求比例/要求人数/当前比例/当前人数/状态│
│   └─ 柱状图: 要求 vs 实际对比                           │
├─────────────────────────────────────────────────────────┤
│ Section 3: 分布规则                                     │
│   └─ 规则卡片列表                                       │
├─────────────────────────────────────────────────────────┤
│ Section 4: 分布预览                                     │
│   └─ 人数输入 + 网格预览（每个等级的人数和比例）         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ Modal: 新建/编辑规则弹窗                                │
└─────────────────────────────────────────────────────────┘
```

## 2. 数据结构

### 2.1 Grade 对象（等级定义）

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 等级名称 |
| `code` | string | 等级代码 |
| `min` | number | 分数下限 |
| `max` | number | 分数上限 |
| `color` | string | CSS 变量名（如 `--em`） |
| `order` | number | 排序序号 |

### 2.2 Dist 对象（分布比例）

| 字段 | 类型 | 说明 |
|------|------|------|
| `grade` | string | 等级名称 |
| `required` | number | 要求百分比 |
| `requiredN` | number | 要求人数（计算值） |
| `current` | number | 当前百分比 |
| `currentN` | number | 当前人数（计算值） |

### 2.3 Rule 对象（分布规则）

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 规则名称 |
| `type` | `'strong' \| 'weak'` | 强控/弱控 |
| `minHead` | number | 最少人数 |
| `groups` | string[] | 适用考核组 |
| `desc` | string | 规则描述 |

## 3. Section 1: 等级定义字段

### 3.1 表格字段

| 列名 | 控件 | 类型 | 必填 | 校验 | 说明 |
|------|------|------|------|------|------|
| 等级名称 | 文本 | string | — | — | 带颜色圆点 |
| 等级代码 | 文本 | string | — | — | 等宽字体 |
| 分数区间(下限) | input number | number | 是 | 0-100 | 小输入框 |
| 分数区间(上限) | input number | number | 是 | 0-100 | 小输入框 |
| 颜色标识 | 文本 | string | — | — | 颜色圆点 + 变量名 |
| 排序 | input number | number | 是 | ≥1 | 小输入框 |
| 操作 | 按钮 | — | — | — | 删除按钮 |

### 3.2 操作

| 操作 | 行为 |
|------|------|
| 添加等级 | 在表格末尾新增一行 |
| 删除等级 | 直接删除（无确认） |
| 编辑分数区间 | 实时更新分布比例 |

## 4. Section 2: 分布比例配置字段

### 4.1 表格字段

| 列名 | 控件 | 类型 | 说明 |
|------|------|------|------|
| 等级名称 | 文本 | string | |
| 要求比例 | input number | number | 可编辑，后缀"%" |
| 要求人数 | 文本 | number | 计算值 |
| 当前比例 | 文本 | number | 百分比 |
| 当前人数 | 文本 | number | |
| 状态 | badge | — | 正常(绿)/接近(橙)/超出(红) |

### 4.2 合计行

| 字段名 | 说明 |
|--------|------|
| 合计要求比例 | 必须等于 100%，否则红色 |
| 合计要求人数 | 计算值 |
| 合计当前比例 | |
| 合计当前人数 | |
| 校验状态 | "校验通过"(绿) / "比例不为100%"(红) |

### 4.3 状态判定逻辑

```
diff = current - required
if diff > 3 → badge-err（"超出"）
if diff > 0 → badge-warn（"接近"）
else → badge-ok（"正常"）
```

### 4.4 柱状图字段

| 字段名 | 说明 |
|--------|------|
| 等级名称 | 左侧标签 |
| 要求条 | 半透明，显示要求百分比 |
| 实际条 | 实色，显示当前百分比，颜色按状态变化 |
| 图例 | 要求分布 / 实际分布 |

## 7. 新建/编辑规则弹窗字段

### 7.1 弹窗结构

| 区域 | 内容 |
|------|------|
| 标题栏 | "新建分布规则" / "编辑分布规则" + 关闭按钮 |
| 表单区 | 规则配置字段 |
| 底部按钮 | 取消 / 保存 |

### 7.2 表单字段

| 字段名 | 控件 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| 规则名称 | input text | string | 是 | |
| 规则类型 | radio group | enum | 是 | 强控/弱控（切换按钮样式） |
| 最少人数 | input number | number | 是 | |
| 适用考核组 | checkbox group | string[] | 是 | 多选，可选考核组列表 |
| 规则描述 | textarea | string | 否 | |

### 7.3 规则类型选项

| 值 | 显示 | 样式 |
|----|------|------|
| strong | 强控 | 红色标签，超出时阻止提交 |
| weak | 弱控 | 橙色标签，超出时仅提醒 |

## 8. 提交校验规则

| 校验项 | 规则 | 错误提示 | 阻断 |
|--------|------|----------|------|
| 分布比例合计 | 所有等级要求比例之和必须等于 100% | "比例不为100%" | 是 |
| 分数区间 | 下限 < 上限 | — | 是 |
| 分数区间 | 范围 0-100 | — | 是 |
| 排序 | ≥ 1 | — | 是 |

## 9. 页面交互逻辑汇总

### 9.1 基础操作

| 场景 | 行为 |
|------|------|
| 添加等级 | 表格末尾新增空行 |
| 删除等级 | 直接删除 |
| 编辑分数区间 | 实时更新分布比例和柱状图 |
| 编辑要求比例 | 实时更新要求人数和柱状图 |
| 输入预览人数 | 实时刷新网格预览 |
| 新建规则 | 打开弹窗 → 填写 → 保存 |
| 编辑规则 | 打开弹窗 → 修改 → 保存 |
| 删除规则 | 确认后删除 |

### 9.2 实时联动

| 操作 | 联动更新 |
|------|----------|
| 编辑分数区间 | 分布比例表重新计算 → 柱状图重新渲染 |
| 编辑要求比例 | 要求人数重新计算 → 合计行校验 → 柱状图重新渲染 |
| 添加/删除等级 | 分布比例表增减行 → 柱状图增减条目 → 合计行重算 |
| 输入预览人数 | 网格预览各等级人数实时刷新 |

### 9.3 操作反馈

| 操作 | 反馈方式 | 内容 |
|------|----------|------|
| 添加等级 | 无 toast | 表格末尾新增空行 |
| 删除等级 | 无确认（直接删除） | 行消失 |
| 比例合计 ≠ 100% | 底部红色提示 | "比例不为100%" |
| 比例合计 = 100% | 底部绿色提示 | "校验通过" |
| 保存规则成功 | toast + 弹窗关闭 | "规则已保存" |
| 保存规则失败 | toast（红色） | 具体错误信息 |

### 9.4 异常态交互

| 异常场景 | UI 表现 | 用户操作 |
|----------|---------|----------|
| 分数区间下限 ≥ 上限 | 输入框红色边框 | 修正区间 |
| 分数超出 0-100 范围 | 输入框红色边框 | 修正数值 |
| 排序 < 1 | 输入框红色边框 | 修正排序 |
| 比例合计 ≠ 100% | 底部红色提示 + 保存阻断 | 调整比例 |
| 网络断开保存 | toast 红色"保存失败" | 重试 |

### 9.5 Loading 态

| 场景 | 表现 |
|------|------|
| 页面初始化 | 表格区域显示骨架屏 |
| 保存中 | 按钮显示 spinner + 禁用 |
