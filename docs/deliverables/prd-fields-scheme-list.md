> 基于 `perf-scheme-list.html` 原型提取，版本 v3 快照

---

## 1. 页面布局

```
┌─────────────────────────────────────────────────────────┐
│ Header: Logo + 返回导航 + 标题"绩效方案" + 操作按钮     │
├─────────────────────────────────────────────────────────┤
│ 统计卡片（4列）: 方案总数/已启用/已停用/草稿            │
├─────────────────────────────────────────────────────────┤
│ 筛选栏: 搜索框 + 周期筛选 + 状态筛选 + 重置按钮        │
├─────────────────────────────────────────────────────────┤
│ 方案表格: 名称/编号/周期/状态/活动数/进度/创建人/修改时间/操作 │
├─────────────────────────────────────────────────────────┤
│ 分页栏: 信息 + 分页导航                                 │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────┘
```

## 2. 数据结构

### Scheme 对象

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 方案ID |
| `name` | string | 方案名称 |
| `code` | string | 方案编号 |
| `cycle` | string | 周期类型：月度/半年度/年度/自定义 |
| `status` | `'enabled' \| 'disabled' \| 'draft'` | 启用状态 |
| `groups` | number | 考核组数量 |
| `activities` | number | 关联活动数 |
| `progress` | number | 配置进度百分比 |
| `creator` | string | 创建人 |
| `modified` | string | 最近修改时间 |

### 状态映射

| status | 显示文本 | CSS 类 |
|--------|----------|--------|
| enabled | 已启用 | `.enabled`（绿色） |
| disabled | 已停用 | `.disabled`（灰色） |
| draft | 草稿 | `.draft`（橙色） |

## 4. 筛选栏字段

| 控件 | 类型 | 说明 |
|------|------|------|
| 搜索框 | input text | 按方案名称或编号模糊搜索 |
| 周期筛选 | select | 全部周期/月度/半年度/年度/自定义 |
| 状态筛选 | select | 全部状态/已启用/已停用/草稿 |
| 重置按钮 | btn-s | 清空所有筛选条件 |

### 交互逻辑

- 搜索输入实时过滤（oninput）
- 下拉选择即时生效（onchange）
- 重置按钮清空所有筛选并重新渲染

### 5.2 数据行字段

| 字段名 | 控件 | 数据源 | 说明 |
|--------|------|--------|------|
| 方案名称 | 文本 | scheme.name | 加粗 |

| 关联活动 | 文本 | scheme.activities | 格式"N 个活动"，数字蓝色 |
| 配置进度 | 进度条+文本 | scheme.progress | 进度条颜色：≥80%绿/≥50%橙/<50%红 |
| 创建人 | 文本 | scheme.creator | |
| 最近修改 | 文本 | scheme.modified | 等宽字体，11px |
| 操作 | 按钮组 | — | 编辑/复制/详情/启停/删除 |

### 5.3 操作按钮

| 操作 | 图标 | 行为 |
|------|------|------|
| 编辑 | edit | 跳转到向导编辑页 |
| 复制 | copy | 弹出复制弹窗 |
| 详情 | eye | 跳转到详情页 |
| 启用/停用 | toggle | 切换启用状态 |
| 删除 | trash | 弹出删除确认弹窗 |

### 5.4 进度条颜色规则

```
progress >= 80% → high（绿色 --em）
progress >= 50% → mid（橙色 --ac）
progress < 50%  → low（红色 --ro）
```

## 10. 页面交互逻辑汇总

### 10.1 基础操作

| 场景 | 行为 |
|------|------|
| 搜索 | 实时过滤方案列表 |
| 切换周期筛选 | 即时过滤 |
| 切换状态筛选 | 即时过滤 |
| 重置筛选 | 清空所有条件，重新渲染 |
| 新建方案 | 跳转到向导页 |
| 从模板创建 | 跳转到向导页（预设模板） |
| 编辑 | 跳转到向导编辑页 |
| 复制 | 弹窗 → 输入名称 → 确认后复制 |
| 查看详情 | 跳转到详情页 |
| 启用/停用 | 弹窗确认 → 切换状态 |
| 删除 | 弹窗确认 → 删除方案 |

### 10.2 操作反馈

| 操作 | 反馈方式 | 内容 |
|------|----------|------|
| 搜索 | 无 toast（实时过滤） | 表格立即更新 |
| 新建方案 | 页面跳转 | 跳转到向导页 |
| 复制成功 | toast + 列表刷新 | "方案已复制" |
| 启用成功 | toast + 状态标签更新 | "方案已启用" |
| 停用成功 | toast + 状态标签更新 | "方案已停用" |
| 删除成功 | toast + 行消失 | "方案已删除" |
| 删除失败（已启用） | toast（红色） | "已启用方案不可删除" |

### 10.3 异常态交互

| 异常场景 | UI 表现 | 用户操作 |
|----------|---------|----------|
| 筛选结果为空 | 表格居中提示 | "暂无匹配的绩效方案" |
| 网络断开操作 | toast 红色"操作失败" | 重试 |
| 删除已启用方案 | toast 红色"已启用方案不可删除" | 先停用再删除 |

### 10.4 Loading 态

| 场景 | 表现 |
|------|------|
| 页面初始化 | 表格区域显示骨架屏 |
| 筛选中 | 表格上方显示 mini loading |
| 复制/删除中 | 对应操作按钮显示 spinner |

---

## 11. 空状态

当筛选结果为空时，表格显示居中提示："暂无匹配的绩效方案"。
