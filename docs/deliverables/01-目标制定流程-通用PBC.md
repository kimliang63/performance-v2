# 流程1：通用PBC目标制定流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 通用PBC目标制定流程 |
| 流程编码 | flow_goal_setting_pbc |
| 流程说明 | 员工手动填写PBC目标，经间接上级→直接上级审批后锁定 |
| 适用场景 | 员工自主制定绩效目标（非导入） |

### 流程节点

```
开始 → 目标制定（被考核人）→ 间接上级审批（可跳过）→ 直接上级审批 → 结束
```
