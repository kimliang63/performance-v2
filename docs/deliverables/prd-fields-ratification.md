## 2. 数据结构

### 2.1 Employee 对象（审定员工）

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 姓名 |
| `dept` | string | 部门 |
| `title` | string | 职位 |
| `score` | number | 考核得分（加权总分） |
| `grade` | string | 考核评定的当前等级 |
| `adjusted` | string | 审定调整后的等级 |
| `changed` | boolean | 是否已被调整（adjusted !== grade） |
| `reason` | string | 调整原因（仅调整时有值） |
| `status` | `'已提交' \| '未提交' \| '待提交'` | 提交状态 |

### 2.2 Distribution 对象（强制分布规则）

| 字段 | 类型 | 说明 |
|------|------|------|
| `grade` | string | 等级名称 |
| `required` | number | 要求百分比上限 |
| `requiredN` | number | 要求人数上限 |
| `current` | number | 当前百分比（计算值） |
| `currentN` | number | 当前人数（计算值） |

### 2.3 等级列表

```
const GRADES = ['远超预期','符合预期+','符合预期','符合预期-','低于预期'];
```

## 4. 强制分布统计条字段

位于表格上方，横向展示 5 个等级的分布情况。

### 4.1 统计条结构

| 区域 | 内容 | 说明 |
|------|------|------|
| 标题栏 | 图标 + "强制分布统计" + 状态 badge | badge: ok/warn/error |
| 5列网格 | 每列一个等级 | 见下方字段 |

### 4.2 单个等级列字段

| 字段名 | 控件 | 说明 |
|--------|------|------|
| 等级名称 | 文本 | 如"远超预期" |
| 要求上限 | 文本 | 格式："≤N"（要求人数上限） |
| 实际人数 | 文本 | 当前该等级人数，带状态色（ok/warn/error） |
| 进度条 | bar | 底色=要求百分比，前景色=实际百分比 |

### 4.3 状态判定逻辑

```
distStatus(requiredN, actual):
  ratio = actual / requiredN
  if ratio > 1  → error（超出限制）
  if ratio > 0.8 → warn（接近限制）
  else → ok（符合要求）
```

### 4.4 整体状态

```
getOverallDistStatus():
  遍历所有等级：
    if 任一等级为 error → error（"超出限制"）
    if 任一等级为 warn  → warn（"接近限制"）
    else → ok（"符合要求"）
```

- 点击标签切换 `statusFilter`，表格只显示匹配状态的员工

- 切换后重新渲染表格，不影响分布统计

## 10. 批量导入

## 11. 提交校验规则

### 11.1 校验流程

```
batchSubmit():
  1. 调用 checkDistribution() 获取警告列表
  2. if 警告列表不为空 → 弹出 Warning Modal
  3. else → 直接提交成功
```

### 11.2 强制分布校验

| 校验项 | 规则 | 处理 |
|--------|------|------|
| 等级人数上限 | 实际人数 > 要求人数上限 | 生成警告信息 |
| 多等级超限 | 可能多个等级同时超限 | 列出所有超限等级 |

### 11.3 Warning Modal 字段

| 区域 | 内容 |
|------|------|
| 标题 | "强制分布校验" |
| 提示文本 | "以下等级超出强制分布限制：" |
| 警告列表 | 每条格式："{等级}：要求≤{N}人，当前{M}人，超出{M-N}人" |
| 说明文本 | "强制控制模式下，超出限制的提交将被阻止。弱控制模式下可继续提交但需确认。" |
| 返回修改按钮 | btn-s，关闭弹窗 |
| 确认提交按钮 | btn-w，强制提交（弱控制模式） |

### 11.4 校验时机

| 场景 | 行为 |
|------|------|
| 批量确认 | 先校验分布 → 超限弹窗 → 确认则强制提交 |
| 弱控制模式 | 弹窗后可选择"确认提交"绕过限制 |
| 强制控制模式 | 超限则阻止提交（原型未实现此分支） |

## 12. 页面交互逻辑汇总

### 12.1 基础操作

| 场景 | 行为 |
|------|------|
| 点击员工姓名 | 展开/收起详情行（显示原始等级、调整后等级、调整原因） |
| 切换调整后等级 | select onchange → 更新 adjusted 值 → 自动计算 changed 标记 → 重新渲染 |
| 点击状态标签 | 切换 statusFilter → 表格只显示匹配状态的员工 |
| 模板导出 | 生成 CSV 文件下载 |
| 批量导入 | 打开导入弹窗 → 选择文件 → 预览 → 确认导入 |
| 重置所有 | 还原 employees 到初始状态 → 清空展开行 → 重新渲染 |
| 批量确认 | 校验分布 → 超限弹窗 → 确认提交 |
| 弹窗返回修改 | 关闭 Warning Modal |
| 弹窗确认提交 | 关闭 Modal → alert 提交成功 |
| 分布统计实时更新 | 每次等级调整后自动重新计算分布并更新统计条和面板 |

### 12.2 状态驱动 UI

| 状态 | UI 表现 |
|------|---------|
| `pending_direct_rate`（直接上级评级） | 直接上级可编辑"调整后等级"；间接上级/逐级上级不可见 |
| `pending_indirect_rate`（间接上级评级） | 间接上级可编辑"调整后等级"；直接上级的调整结果只读展示 |
| `pending_hierarchy_rate`（逐级上级评级） | 逐级上级可编辑"调整后等级"；前序审批结果只读展示 |
| `pending_hrbp_approval`（HRBP审批） | HRBP 全部只读查看；仅可操作"同意/不同意" |
| `completed` | 所有字段只读；显示绿色"已审定"状态标签 |

### 12.3 强制分布面板联动

| 操作 | 面板更新 |
|------|----------|
| 调整某员工等级 | 实时重新计算各等级人数和比例 → 更新分布统计条 → 更新柱状图 |
| 强控超限 | 对应等级柱状图变红；提交弹窗阻止并返回修改 |
| 弱控超限 | 对应等级柱状图变橙；允许提交但附带警告 |
| 全部等级比例合计 ≠ 100% | 底部显示红色"比例不为100%"提示 |

### 12.4 操作反馈

| 操作 | 反馈方式 | 内容 |
|------|----------|------|
| 调整等级 | 无 toast（实时更新） | 统计面板数字和图表实时变化 |
| 批量确认成功 | toast + 状态变更 | "审定已提交" |
| 强控超限提交 | 弹窗 | "强制分布校验未通过：{等级}超出限制，请调整后重新提交" |
| 弱控超限提交 | 弹窗（可强制提交） | "以下等级超出限制：{列表}，确认提交？" |
| 导入成功 | toast | "成功导入 N 条" |
| 导入部分失败 | 弹窗明细 | "成功 N 条，失败 M 条" + 失败行号和原因 |
| HRBP 审批通过 | toast + 状态变更 | "HRBP 已审批通过" |
| HRBP 驳回 | toast + 状态回退 | "HRBP 驳回，需重新审定" |

### 12.5 跨页面联动

| 触发事件 | 联动行为 |
|----------|----------|
| 审定完成（`completed`） | 员工确认表单自动创建，审定结果写入确认表单只读区 |
| HRBP 驳回 | 审定流程回到 `pending_direct_rate`，直接上级收到通知 |
| 跳过审定的员工 | 使用方案配置的固定等级，计入强制分布 |

### 12.6 异常态交互

| 异常场景 | UI 表现 | 用户操作 |
|----------|---------|----------|
| 网络断开提交 | toast 红色"提交失败" | 数据保留 |
| 并发编辑（两人同时审定同一员工） | 弹窗"该表单已被其他审批人处理，请刷新" → 强制刷新 | 查看最新状态 |
| 强控死锁（所有等级满额） | 弹窗"强制分布已满，请调整其他员工等级" → 提交阻止 | 先调整其他员工 |
| 分布比例合计 ≠ 100% | 底部红色提示 | 调整比例 |
| 导入文件格式错误 | 弹窗提示 | 重新选择文件 |
| 导入数据校验失败 | 弹窗明细 | 修正后重新导入 |

### 12.7 Loading 态

| 场景 | 表现 |
|------|------|
| 页面初始化 | 表格区域显示骨架屏 |
| 等级调整中 | 统计面板显示 mini loading |
| 保存/提交中 | 按钮显示 spinner + 禁用 |
| 导入处理中 | 上传区显示进度条 |
