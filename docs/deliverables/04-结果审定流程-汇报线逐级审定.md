# 流程4：汇报线逐级审定

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 汇报线逐级审定 |
| 流程编码 | flow_result_determination_reporting_line |
| 流程说明 | 按汇报关系逐级审定绩效等级，到特定高管结束 |
| 适用场景 | 汇报线审定模式（用户在绩效方案配置时勾选） |

### 流程节点

```
开始 → 直接上级评级 → 间接上级评级（可跳过）→ 逐级上级评级（可跳过）→ HRBP审批（仅远超预期）→ 结束
```
