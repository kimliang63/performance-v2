# 流程7：绩效面谈流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 绩效面谈流程 |
| 流程编码 | flow_interview |
| 流程说明 | 员工填写面谈记录，直接上级审批确认 |
| 适用场景 | 绩效面谈环节 |

### 流程节点

```
开始 → 本人填写 → 直接上级审批 → 结束
```
