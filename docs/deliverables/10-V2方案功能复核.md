## 六、方案与活动的关系

### 配置层级

```
绩效方案（V2配置）          活动实例化（V4执行）
┌──────────────────┐      ┌──────────────────┐
│ 考核组            │ copy→│ 参与人员（可微调） │
│ 环节配置          │ copy→│ 环节配置（可修改） │
│ 流程模板          │ copy→│ 流程模板          │
│ 表单模板          │ copy→│ 表单模板          │
│ 强制分布          │ copy→│ 强制分布（可修改） │
│ 评价规则          │ copy→│ 评价规则          │
│ 节点权重          │ copy→│ 节点权重（可修改） │
└──────────────────┘      └──────────────────┘
                                │
                                ▼
                          最终按活动配置执行
```

### 核心规则

| 规则 | 说明 |
|------|------|
| 活动实例化时 | copy方案的全部配置 |
| 活动可修改 | 参与人员、环节配置、强制分布、节点权重均可在活动级调整 |
| 执行时 | 只看活动配置，不回溯方案 |
| 方案停用 | 已发起的活动不受影响，继续按活动配置执行 |
