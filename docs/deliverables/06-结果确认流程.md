# 流程6：通用确认+申诉流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 通用确认+申诉流程 |
| 流程编码 | flow_result_confirm |
| 流程说明 | 员工确认绩效结果，不同意时进入HRBP审批 |
| 适用场景 | 绩效审定完成后的结果确认环节 |

### 流程节点

```
开始 → 本人确认 → [不同意] → HRBP知悉 → 结束
                  [同意] → 结束
```
