# 方案详情 — 字段清单与交互规则

> 基于 `perf-scheme-detail.html` 原型提取，版本 v3 快照

---

## 1. 页面布局

```
┌─────────────────────────────────────────────────────────┐
│ Header: Logo + 返回导航                                  │
├──────────┬──────────────────────────────────────────────┤
│ 侧边栏   │ 内容区                                       │
│ (220px)  │ 按 activeSection 渲染：                      │
│          │  ├─ basic → 方案基本信息（只读）             │
│ Tab 切换  │  ├─ grades → 等级规则定义（只读表格）        │
│ ├ 方案信息│  ├─ stages → 考核环节开关                   │
│ ├ 关联活动│  ├─ activities → 关联活动列表               │
│          │  ├─ g1/g2/... → 考核组详情                 │
│ 配置目录 │  └─ cfg-goal/eval/... → 环节配置详情       │
│ ├ 基本信息│                                             │
│ ├ 等级规则│                                             │
│ ├ 考核环节│                                             │
│ └ 考核组×N│                                             │
│   └ 环节  │                                             │
└──────────┴──────────────────────────────────────────────┘
```

## 2. 数据结构

### SCHEME 对象

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 方案名称 |
| `code` | string | 方案编号 |
| `cycle` | string | 周期类型 |
| `enabled` | boolean | 启用状态 |
| `downward` | boolean | 向下公开 |
| `desc` | string | 描述 |
| `creator` | string | 创建人 |
| `created` | string | 创建时间 |
| `modified` | string | 最近修改时间 |

## 5. 关联活动 Tab

### 5.1 活动列表

| 字段名 | 控件 | 数据源 | 说明 |
|--------|------|--------|------|
| 活动名称 | 链接文本 | activity.name | 点击跳转到活动详情 |
| 考核周期 | 标签 | activity.cycle | 蓝底标签 |
| 状态 | 状态标签 | activity.status | 进行中=绿/已结束=灰/未开始=橙 |
| 开始日期 | 文本 | activity.startDate | 等宽字体 |
| 结束日期 | 文本 | activity.endDate | 等宽字体 |
| 参与人数 | 数字 | activity.participants | 蓝色数字 |
| 进度 | 进度条+文本 | activity.progress | ≥80%绿/≥50%橙/<50%红 |
| 操作 | 按钮组 | — | 查看详情 / 发起活动 |

### 5.2 活动状态映射

| status | 显示文本 | 颜色 |
|--------|----------|------|
| active | 进行中 | 绿色 |
| completed | 已结束 | 灰色 |
| not_started | 未开始 | 橙色 |

### 5.3 空状态

当方案无关联活动时，显示居中提示："暂无关联考核活动" + [发起活动] 按钮。

## 8. 页面交互逻辑汇总

### 8.1 基础操作

| 场景 | 行为 |
|------|------|
| 点击侧边栏目录项 | 切换 activeSection → 渲染对应内容 |
| 查看基本信息 | 只读展示方案配置 |
| 查看等级规则 | 只读表格展示 |
| 查看考核环节 | 列表展示，可点击进入详情 |
| 查看考核组 | 只读展示考核组配置 |
| 查看环节配置 | 只读展示环节配置参数 |

### 8.2 状态驱动 UI

| 方案状态 | UI 表现 |
|----------|---------|
| `draft` | 显示"编辑"和"删除"按钮 |
| `enabled` | 显示"编辑"和"停用"按钮；隐藏"删除" |
| `disabled` | 显示"编辑"和"启用"按钮；隐藏"删除" |

### 8.3 异常态交互

| 异常场景 | UI 表现 | 用户操作 |
|----------|---------|----------|
| 方案不存在 | 页面显示"方案不存在"提示 | 返回列表 |
| 网络断开加载 | 页面显示"加载失败" + 重试按钮 | 点击重试 |

### 8.4 Loading 态

| 场景 | 表现 |
|------|------|
| 页面初始化 | 各区域显示骨架屏 |
| 切换侧边栏 | 内容区短暂 loading |
