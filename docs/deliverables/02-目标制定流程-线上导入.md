# 流程2：线上导入目标制定流程

## 一、流程概述

| 项目 | 内容 |
|------|------|
| 流程名称 | 线上导入目标制定流程 |
| 流程编码 | flow_goal_setting_import |
| 流程说明 | HRBP批量导入员工目标，员工确认后经间接上级→直接上级审批锁定 |
| 适用场景 | 批量导入绩效目标（Excel导入） |

### 流程节点

```
开始 → 目标导入（HRBP批量）→ 员工确认 → 间接上级审批（可跳过）→ 直接上级审批 → 结束
```

### 流程引擎兼容问题

> **已知问题**：目标导入为批量处理节点，多人同时流入HRBP时，流程引擎需要解决：
> 1. **业务层判断**：如何识别批量节点，呈现批量处理页面给HRBP
> 2. **通知平衡**：批量节点不断流入新员工时，如何避免重复提醒，同时保证新待办能被感知
> 3. **批量表单**：使用对应的批量表单页面，而非单人表单
>
> 此问题需流程引擎侧提供解决方案，作为V3流程引擎的待解决问题。
