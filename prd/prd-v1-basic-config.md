## 1. 绩效周期

### 功能描述
设置绩效周期规则，定义考核的时间范围和类型。

### 功能入口
GHR / 绩效管理 / 绩效基础配置 / 绩效周期

### 菜单名称/页签名称
绩效周期

### 数据权限管控维度
无

### 多语言环境显示
按通用规范处理

### 功能权限

| 菜单 | 权限项 | 说明 |
|------|--------|------|
| 绩效周期 | 查询 | 控制【绩效周期】页面的查询，无权限则显示无权限缺省页 |
| | 新增 | 控制【绩效周期】页面的新增按钮 |
| | 编辑 | 控制【绩效周期】页面的编辑按钮 |
| | 启用/停用 | 控制【绩效周期】页面的启用/停用按钮 |
| | 删除 | 控制【绩效周期】页面的删除按钮 |

### 功能说明

#### 1.1 列表

**列表字段**

| 字段 | 类型 | 说明 |
|------|------|------|
| 名称 | 多语言文本 | 周期名称 |
| 类型 | 枚举 | 固定周期（年度/半年度/季度/月度）/ 非固定周期 |
| 开始日期 | 日期 | 归属年份 + 日期 |
| 结束日期 | 日期 | 归属年份 + 日期 |
| 系统预置 | 布尔 | 是否为系统预置数据 |
| 是否启用 | 枚举 | 启用/停用 |
| 最近修改人 | 文本 | 最近修改的操作人 |
| 最近修改时间 | 时间 | 最近修改时间 |

**排序**
- 默认按"排序码"升序排序

**查询**

| 字段 | 控件类型 | 说明 |
|------|----------|------|
| 名称 | 文本框 | 默认为空，提示"请输入"，模糊搜索绩效周期名称 |

**筛选**

| 字段 | 控件类型 | 说明 |
|------|----------|------|
| 类型 | 多选框 | 默认为空，提示"请选择"，展示所有类型选项 |
| 系统预置 | 多选框 | 默认为空，提示"请选择"，展示启用/停用选项 |
| 启用 | 多选框 | 默认为空，提示"请选择"，展示启用/停用选项 |

**操作**

| 操作 | 说明 | 前置条件 |
|------|------|----------|
| 新增 | 点击跳转新增/编辑页面，页面状态为默认 | 有新增权限 |
| 编辑 | 点击跳转新增/编辑页面，加载目标数据 | 有编辑权限 |
| 启用/停用 | 点击更新当前数据状态为启用/停用 | 有启用/停用权限 |
| 删除 | 弹出二次确认框，确认后删除 | 有删除权限，且状态为停用 |

**删除校验规则**

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| 状态校验 | 仅停用状态可删除 | "仅停用状态可删除" |
| 引用校验 | 检查是否有进行中/已完成的绩效活动关联 | "周期已被引用不可删除" |

#### 1.2 新增/编辑

**表单字段**

| 字段 | 控件类型 | 必填 | 校验规则 | 说明 |
|------|----------|------|----------|------|
| 名称 | 文本框 | 是 | 最多100字 | 支持多语言（小地球按钮） |
| 是否启用 | 单选框 | 是 | 默认选中"启用" | 选项：启用/停用 |
| 排序码 | 文本框 | 是 | 仅可输入正整数 | 用于排序 |
| 类型 | 单选框 | 是 | 默认为空 | 固定周期（年度/半年度/季度/月度）/ 非固定周期 |
| 归属年份（开始） | 单选框 | 是 | 默认选中"本年" | 本年/上一年/下一年 |
| 开始日期 | 日期选择器 | 条件必填 | 固定周期时必填 | 选择日期后回显 MM/DD |
| 归属年份（结束） | 单选框 | 是 | 默认选中"本年" | 本年/上一年/下一年 |
| 结束日期 | 日期选择器 | 条件必填 | 固定周期时必填 | 选择日期后回显 MM/DD |

**字段交互逻辑**

| 触发条件 | 交互行为 |
|----------|----------|
| 类型选择"非固定周期" | 隐藏开始日期、结束日期相关字段 |
| 类型选择"固定周期" | 显示开始日期、结束日期相关字段 |

**提交校验规则**

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| 必填项校验 | 名称、排序码、类型必填 | "必填项不能为空" |
| 名称唯一性 | 校验各语种名称是否唯一 | "{语种}已存在名称，请修改" |
| 排序码唯一性 | 校验排序码是否唯一 | "已存在排序码，请修改" |
| 日期规则 | 归属年份均为本年时，开始日期不可大于等于结束日期 | "开始日期不能大于等于结束日期" |
| 日期规则 | 开始日期归属年份不可大于结束日期归属年份 | "开始日期归属年份不能大于结束日期归属年份" |

**后端处理逻辑**

```
1. A 校验
   - 必填项校验
   - 名称唯一性校验（各语种）
   - 排序码唯一性校验
   - 日期规则校验

2. B 处理
   - 生成周期编码（系统自动生成）
   - 设置系统预置标志为false

3. C 存储
   - 新增：INSERT INTO perf_cycle
   - 编辑：UPDATE perf_cycle SET ... WHERE id = ?

4. D 返回
   - 返回操作结果
   - 提示"操作成功"
   - 跳转上级页面
```

```

   - 名称唯一性校验（各语种）
   - 等级名称唯一性校验（各语种）

   - 生成评价规则编码（系统自动生成，前缀"PESC"）

4. D 返回
   - 返回操作结果
   - 提示"操作成功"
   - 跳转上级页面
```
