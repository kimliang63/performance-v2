## 原型页面规划

### perf-activity.html — 活动列表
- 活动卡片列表（状态标签、进度、操作按钮）
- 发起活动弹窗（选择方案→配置周期→设置名称→选择时间）
- 筛选：状态、方案、周期
- 操作：查看、编辑、终止、删除

### perf-activity-detail.html — 活动详情
- 顶部信息栏（活动名称、状态、关联方案、操作按钮）
- 4个Tab：
  - 总览：统计卡片 + 进度条 + 分组明细
  - 考核人详情：人员列表 + 各环节状态 + 操作
  - 操作日志：操作记录表格
  - 设置：活动配置

### perf-ai-monitor.html — AI 绩效活动监控
- 顶部信息栏 + AI助手按钮
- 4个Tab：
  - 总览：统计卡片 + AI风险预警 + 活动进度
  - 风险管理：风险事项表格 + 批量操作
  - 智能催办：催办统计 + 催办记录
  - 操作日志：日志表格
