# 绩效管理 V1：绩效基础配置

## 版本目标
搭建绩效基础配置能力，包括周期、单位、等级、计算规则、衡量标准、指标库、评价规则、强制分布规则。

## 状态标记
标注"已评审"的功能已完成需求评审。

## 功能清单

### 绩效周期（已评审）
- 入口：GHR / 绩效管理 / 绩效基础配置 / 绩效周期
- 类型：固定周期（年度/半年度/季度/月度）、非固定周期
- 操作：新增/编辑/启用/停用/删除
- 排序码管理
- 系统预置数据

### 单位（已评审）
- 入口：GHR / 绩效管理 / 绩效基础配置 / 单位
- 操作：新增/编辑/删除
- 排序：按最近修改时间降序

### 等级规则（已评审）
- 入口：GHR / 绩效管理 / 绩效基础配置 / 等级规则
- 字段：名称、部门、向下公开、匹配规则、等级名称、左右区间、分值
- 支持多等级配置

### 计算规则
- 入口：GHR / 绩效管理 / 绩效基础配置 / 计算规则
- 公式编辑器：沿用考勤公式平台
- 编码前缀：PERSC
- 支持导入/导出
- 公式校验：语法/引用权限/类型

### 衡量标准
- 入口：GHR / 绩效管理 / 绩效基础配置 / 衡量标准
- 类型：固定值（未来拓展更多类型）
- 支持梯度配置：梯度名称、梯度规则（整数/小数）

### 指标库
- 入口：GHR / 绩效管理 / 绩效基础配置 / 指标库
- 树控件：指标分类管理
- 指标类型：定性指标、定量指标、加减法指标
- 指标性质：正向指标、负向指标
- 关联：计算规则、单位、衡量标准
- 完成值来源：用户填写（未来拓展）
- 支持导入/导出

### 评价规则
- 入口：GHR / 绩效管理 / 绩效基础配置 / 评价规则
- 规则类型：数值、等级
- 分值范围：最小值~最大值
- 编码前缀：PESC

### 强制分布规则
- 入口：GHR / 绩效管理 / 绩效基础配置 / 强制分布规则
- 应用等级选择
- 规则类型：数值、等级
- 生效人数配置
- 计算规则：不超过、不低于
