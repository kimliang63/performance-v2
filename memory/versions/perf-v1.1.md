# 绩效管理 V1.1：公式编辑器

## 版本目标
独立的公式编辑器能力，支持数据源定义、计算元素、表达式编辑与校验。

## 应用流程
1. 通用层维护（初始化）：数据源定义 → 计算科目维护 → 业务应用科目授权
2. 业务实时计算：调用计算 → 解析表达式 → 权限校验 → 查询数据 → 执行计算

## 核心能力

### 数据源定义
- 数据源定义版本管理

### 计算科目
- 静态科目(BOUND)：绑定源字段
- 动态科目(DYNAMIC)：配置动态取数规则

### 表达式编辑器
- 样式参考UI设计稿
- 应用方式：弹窗、页面嵌套
- 科目过滤规则：应用白名单 + 动态数据源级可见/可用（取交集）
- 操作：添加字段/函数/运算符、清空、检查、保存/提交、取消
- 快捷键：ctrl+z（撤回10步）、ctrl+c/v（复制粘贴）

### 表达式校验
- 校验时机：保存/提交、试算
- Error类错误不允许保存

### 表达式解析与执行
- 语法解析 → 批次队列逐条执行 → 结果记录日志
- 错误兜底：默认返回"null"
- 日志记录：执行批次、执行时间、执行结果明细
